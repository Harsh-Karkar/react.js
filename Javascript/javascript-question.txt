JavaScript એ હળવા વજનની, ક્રોસ-પ્લેટફોર્મ, સિંગલ-થ્રેડેડ અને અર્થઘટન કરેલ સંકલિત પ્રોગ્રામિંગ ભાષા છે. તે વેબપૃષ્ઠો માટે સ્ક્રિપ્ટીંગ ભાષા તરીકે પણ ઓળખાય છે. તે વેબ પૃષ્ઠોના વિકાસ માટે જાણીતું છે, અને ઘણા બિન-બ્રાઉઝર વાતાવરણ પણ તેનો ઉપયોગ કરે છે.

JavaScript એ નબળી રીતે ટાઈપ કરેલી ભાષા છે (ડાયનેમિકલી ટાઈપ કરેલી). JavaScript નો ઉપયોગ ક્લાયંટ-સાઇડ વિકાસ તેમજ સર્વર-સાઇડ વિકાસ માટે કરી શકાય છે. JavaScript એ અનિવાર્ય અને ઘોષણાત્મક બંને પ્રકારની ભાષા છે. JavaScript ઑબ્જેક્ટ્સની પ્રમાણભૂત લાઇબ્રેરી ધરાવે છે, જેમ કે એરે, તારીખ અને ગણિત, અને ઓપરેટર્સ, કંટ્રોલ સ્ટ્રક્ચર્સ અને સ્ટેટમેન્ટ્સ જેવા ભાષા ઘટકોનો મુખ્ય સમૂહ.
